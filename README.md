# unity-pixel-game
Juego 2D de la iglesia de quito de la compañía de Jesús un lugar histórico de Quito. Grupo: Patrick Goyez - Alexander Montero - Edward Fizgerald - Matias Romero


Prompts usados: 
Prompt 1: Configuración Inicial y Concepto
"Actúa como un experto en desarrollo de videojuegos en Unity para principiantes. Necesito crear un prototipo de minijuego 2D tipo exploración lateral (side-scroller) sobre la Iglesia de la Compañía de Jesús en Quito. El juego debe tener un estilo visual de vóxeles/Minecraft. Explícame paso a paso cómo instalar Unity, crear un proyecto 2D y diseñar una escena básica con un suelo, una iglesia (formas simples) y un personaje (un cuadrado azul)."

Prompt 2: Movimiento y Física del Jugador
"Dame un script simple en C# para el movimiento de un personaje 2D en Unity. Necesito que se mueva a la izquierda, derecha y que pueda saltar con la barra espaciadora. Explica cómo añadir un Rigidbody2D y un Box Collider 2D para que el personaje tenga gravedad y no atraviese el suelo."

Prompt 3: Sistema de Interacción Histórica
"Necesito crear una puerta interactiva para la iglesia. Dame un script en C# que detecte cuando el jugador está cerca (usando un Trigger). Cuando el jugador presione la tecla 'E', debe aparecer un panel de interfaz (UI) con un texto histórico sobre la construcción de la Iglesia de la Compañía y su arte barroco. El panel debe desaparecer si el jugador se aleja de la puerta."

Prompt 4: Sistema de Coleccionables y Contador
"Agrega una mecánica de recolección al juego. Necesito un script para un objeto llamado 'Rollo Fotográfico'. Cuando el jugador lo toque, el objeto debe desaparecer. Además, crea un script controlador que lleve la cuenta de cuántos rollos se han recogido y muestre un contador (ej: 'Rollos: 0/5') en la pantalla usando un elemento de texto de la UI."

Prompt 5: Cámara y Pulido Visual
"Dame un script simple para que la cámara siga al jugador suavemente mientras camina. Además, dime qué configuraciones debo cambiar en las imágenes (sprites) en el Inspector de Unity (como el Filter Mode y la Compresión) para que el arte se vea pixelado y nítido al estilo Minecraft, no borroso."
